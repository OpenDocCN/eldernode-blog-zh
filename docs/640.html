如何在Kali Linux上安装dex 2 jar

Dex2Jar工具是Kali Linux的逆向工程之一。它预装在逆向工程工具下的 Kali Linux 上。要在Kali Linux上安装它，请使用以下命令:

sudo apt-get install dex2jar

如何将jar转换为dex

android sdk的dx工具可以完成这项工作

dx --dex --output=classes.dex abc.jar

来自dex2jar的[d2j-jar2dex]也可以完成这项工作

d2j-jar2dex.sh --output=classes.dex abc.jar
