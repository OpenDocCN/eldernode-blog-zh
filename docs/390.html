如何在CentOS 8上安装和配置GIMP 2.10

为了让本教程更好地发挥作用，请考虑以下先决条件:

1- 拥有sudo权限的非root用户

2- 进行设置，遵循我们的初始设置centos 8

而 Gimp 包可在安装CentOS/RHEL 8时默认添加的AppStream存储库中获得。

sudo dnf provides gimp

然后，您可以通过运行以下命令在CentOS 8上安装Gimp 2.x:

sudo dnf -y install gimp

接下来，从您的菜单或通过键入以下命令启动Gimp图形编辑器:

gimp

运行以下命令以确认安装并检查软件包详细信息:

dnf info gimp

一旦安装完成，你可以从网页界面或桌面应用程序启动程序，如Gnome，KDE等

如何在CentOS 7上安装GIMP 2.10

你无法想象在CentOS 7上安装GIMP有多容易。但这是因为您只需要一个安装命令。因此，不要再等待，执行以下命令:

sudo yum install gimp

当你看到下面的页面，这意味着一切正常。
