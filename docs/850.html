在AlmaLinux 8.4上安装Nodejs

在这一节，我们将教你如何在AlmaLinux 8.4上安装Nodejs。为此，您必须遵循以下步骤。

第一步是使用以下命令使服务器保持最新状态:

dnf update -y

然后，您应该通过运行以下命令列出可用的Nodejs流。应该注意，该流与Nodejs的原始版本兼容:

dnf module list nodejs

最后，您可以使用以下命令在AlmaLinux上安装Nodejs :

dnf module install nodejs:14
