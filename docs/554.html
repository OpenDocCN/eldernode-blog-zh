如何卸载oVirt-guest-agent

如果您希望从Ubuntu 20.04 中卸载oVirt，您可以使用以下命令删除ovirt-guest-agent包本身。

sudo apt-get remove ovirt-guest-agent

此外，要卸载ovirt-guest-agent及其依赖项，请运行:

sudo apt-get remove --auto-remove ovirt-guest-agent

但是如果您需要删除ovirt-guest-agent的本地/配置文件，请使用下面的命令。注意，与上面两个相反，被清除的配置/数据不能通过重新安装软件包来恢复。

sudo apt-get purge ovirt-guest-agent
