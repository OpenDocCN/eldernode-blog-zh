在Ubuntu 20.04上安装WebSploit | Ubuntu 18.04

在这一节，我们想教你如何在Ubuntu 20.04上安装WebSploit。为此，只需运行以下命令来更新系统:

sudo apt-get update

现在，您可以使用以下命令轻松安装Websploit框架:

sudo apt-get install websploit

成功安装该工具后，只需运行以下命令即可运行它:

websploit
