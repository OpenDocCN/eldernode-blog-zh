教程在Centos/RHEL 8 Linux - Eldernode上安装Python 3.8

原文：https://blog.eldernode.com/install-python-3-8-on-centos/#0001-01-01

在本教程中，我们将一步步介绍如何在Centos / RHEL 8 Linux上安装Python 3.8。

教程如何在Centos/RHEL 8 Linux上配置和安装Python 3.8

使用以下命令安装所需的 Python 开发库。

sudo dnf install gcc openssl-devel bzip2-devel libffi-devel

步骤1)更新百分位T3】

在执行任何操作之前，您需要更新Centos，请使用以下命令:

yum -y update

步骤2)下载Python 3.8

您可以使用以下命令从Python的官方网站下载Python。

cd /opt  wget https://www.python.org/ftp/python/3.8.3/Python-3.8.3.tgz

注意:如果没有启用wget，可以使用以下命令安装它:

yum -y install wget

然后，使用下面的命令，您必须提取下载的归档文件:

tar xzf Python-3.8.3.tgz

步骤3)在Centos上安装Python

解压缩后，转到Python-3.8.3文件夹，然后根据您的服务器或系统环境配置源文件。

cd Python-3.8.3  sudo ./configure --enable-optimizations  sudo make altinstall

步骤4)在Centos 8上测试Python

完成安装步骤后，我们使用以下命令测试已安装的Python版本:

python3.8 -V

输出应该如下所示:

Python 3.8.3

恭喜您，您已经能够在Centos上成功安装和配置Python。
