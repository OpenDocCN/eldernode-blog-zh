下一节，我们将教你如何在 Ubuntu 22.04上安装Steam。

如何在Ubuntu 22.04上安装SteamT3】

首先用下面的命令更新APT包:

sudo apt update

然后运行下面的命令开始安装:

sudo apt install steam-installer
