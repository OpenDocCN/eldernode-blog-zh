CMake在操作系统中以独立于编译器的方式管理构建过程。如果你在 Windows 上从源码树构建CMake，你应该先安装最新的Cmake二进制版本。安装时，在CMake上运行二进制文件。换句话说，当您选择CMake作为源目录时，您就为生成的可执行文件选择了一个二进制目录。
