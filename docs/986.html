如何在Kali Linux上安装Recon-ng

您可以使用下面的命令在Kali Linux上安装Recon-ng :

apt install recon-ng

如何在Kali Linux上使用Recon-ng

在下文中，我们将回顾不同的识别工具以及如何在Kali Linux上使用它们:

在Kali Linux上使用Recon-CLI

这个工具允许从命令行使用Recon-ng。您可以通过以下命令使用此工具:

recon-cli -h

在Kali Linux上使用Recon-ng

如前所述，这个工具是一个网络识别框架。运行以下命令来使用它:

recon-ng -h

在Kali Linux上使用Recon-web

这个工具是一个基于网络的用户界面。通过运行以下命令来使用它:

recon-web -h
