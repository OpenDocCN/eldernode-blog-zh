教程在Ubuntu 20.04上安装SQL inja

Sqlninja是在Gentoo平台上开发的，可以在Linux、FreeBSD、Mac OS X和iOS等操作系统上运行。Sqlninja是一个很棒的SQL注入工具。还有一些其他的工具，如Bobcat、ExploitMyUnion和鸦片酊。Sqlninja需要一些Perl组件。要在Ubuntu中安装Sqlninja，首先需要安装Perl模块。

打开终端并运行以下命令:

perl -MCPAN -e "install Net::RawIP"
perl -MCPAN -e "install Net::Pcap"
perl -MCPAN -e "install Net::PcapUtils"
perl -MCPAN -e "install Net::Packet"
perl -MCPAN -e "install Net::DNS"
perl -MCPAN -e "install IO::Socket::SSL"

然后，您可以下载Sqlninja来使用它。此外，您可以使用下面的命令下载并解压缩Sqlninja文件夹。打开终端并键入:
