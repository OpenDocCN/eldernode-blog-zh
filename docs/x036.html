如何在Windows Server 2019 - Eldernode上设置OpenVPN

原文：https://blog.eldernode.com/setup-openvpn-on-windows-server-2019/#0001-01-01

如果你想保护你的网上隐私或加密你在网上的信息，使其不落入黑客之手；你应该使用VPN。任何VPN的主要元素是它所使用的VPN协议。在本文中，我们将教你如何在Windows Server 2019上安装OpenVpn。大多数VPN支持多种协议，如IKEv2、L2TP和SSTP，但OpenVPN协议被认为是世界上最流行和最好的VPN协议。在这篇文章中，我们将教你如何在Windows Server 2019上设置OpenVPN。如果你想购买自己的，你可以查看 Eldernode 网站上提供的套餐。

教程在Windows Server 2019上设置OpenVPN

什么是OpenVPN？T3】

Openvpn是目前最流行和广泛使用的vpn协议。与其他基于IPSec的隧道协议(如L2tp)不同，Openvpn依赖SSL / TLS进行身份验证和数据加密。这种标准安全技术用于从一个地方到另一个地方或从一点到另一点进行安全的远程连接。使用SSL来保护金融交易、数据传输、电子邮件和其他东西非常流行。

OpenVPN兼容所有主流操作系统，如Windows、Android、iOS、Mac和Linux。它是开源的，支持高达256位的加密。

可用的OpenVPNs如下:

–>OpenVPN社区版:开源免费版，没有用户限制。

–>OpenVPN接入服务器:它基于社区版，但提供额外的付费和专有功能，如轻松管理管理门户、LDAP集成等。

–>OpenVPN-as-a-Service:这是一种无需安装VPN服务器的解决方案。只需购买OpenVPN Cloud，您就可以连接到世界各地的托管服务。

在 Windows培训标签的这一步，你将学习如何在Windows Server 2019上安装OpenVPN。

先决条件

–访问端口1194以连接到同一网络

notepad vars.bat

–>KEY _ CN和KEY_NAME: 它们对于每个构建请求都是唯一的，并且引用通用名称字段和证书的名称。

set KEY_COUNTRY=US set KEY_PROVINCE=CA set KEY_CITY=SanFrancisco set KEY_ORG=OpenVPN set [email protected]

–>KEY _ OU:指的是一个“组织单元”，如果没有要求，可以设置为任意值。

–>PKCS11 _ values:它们是指用于硬件安全模块和智能卡的设置，如果您使用它们的话。

保存文本编辑器文件并退出。

客户端文件修改与服务器配置更改非常相似。
