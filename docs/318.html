教程在DNS Manager Server 2019中创建正向查找区域。如你所知， 转发区 用于将名称转换为IP地址。在本文中，我们将执行创建该区域的设置步骤，如下所示。你可以从eldernode选择你最完美的 Windows VPS服务器包。

 

如何在DNS Manager Server 2019中创建正向查找区域

1.在您的active directory DNS服务器上打开 DNS管理器。

2.在前向查找区上右击，选择新区。

 

 

3.在新区域向导窗口中，点击下一步并继续。

 

4.在下一步中，选择您想要创建的区域的类型，并点击下一步。

 

 

5.选择默认设置并点击确定。

 

 

6.在区域名称字段中，输入您的外部域名。

注:我的外部域名是eldernode.com

 

 

7.因为我们手动创建所有文件，所以在此阶段我们必须单击“不允许动态更新”。

8.点击下一个。

 

9.最后一步，点击 Finish 完成安装过程。
