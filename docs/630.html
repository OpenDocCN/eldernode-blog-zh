在Fedora 33 上安装Mattermost桌面

在本节中，我们使用快照在Fedora 33上安装Mattermost Desktop。快照是打包了所有依赖项的程序，可以在所有流行的Linux发行版中从单个构建运行。要安装Snap，您可以轻松运行以下命令:

sudo dnf install snapd

您可以重启系统一次，以确保应用更改。您可以输入以下命令启用snap 支持，在 /var/lib/snapd/snap 和 /snap 之间创建一个符号链接:

sudo ln -s /var/lib/snapd/snap /snap

最后，您可以轻松地运行以下命令来在Fedora 33上安装Mattermost Desktop :

sudo snap install mattermost-desktop
