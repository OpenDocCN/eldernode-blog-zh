在Windows Server 2016上配置OpenSSLT3】

在您能够成功安装OpenSSL之后，我们现在想要配置它。您可以使用以下命令创建一个测试SSL证书来验证您的安装:

openssl.exe req -new -nodes -keyout server.key -out server.csr -newkey rsa:2048

现在，您可以配置环境变量，使OpenSSL在您的系统上正常工作。这些环境变量是 OPENSSL_CONF 和路径。您可以轻松地使用以下命令进行这些设置:

set OPENSSL_CONF=C:\OpenSSL-Win32\bin\openssl.cfg
set Path=%Path%;C:\OpenSSL-Win32\bin
