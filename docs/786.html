如何在Ubuntu 20.04 | 21.04T3上安装/设置Scribus

在这一节，我们将教你如何一步一步地在Ubuntu 20.04上安装Scribus。为此，只需按顺序执行以下步骤。第一步，你可以通过访问 Scribus AppImage网站下载。请注意，在本教程中，我们将使用官方的PPA安装。

您必须首先使用以下命令添加存储库:

sudo add-apt-repository ppa:scribus/ppa

然后，您可以通过运行以下命令来更新包列表:

sudo apt update

最后，您可以通过运行以下命令轻松地安装Scribus:

sudo apt install scribus
